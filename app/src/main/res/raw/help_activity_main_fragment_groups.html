Groups

This screen lists the groups you've set up, and tapping on one of the groups will let you add or remove friends to the group. You can also create new groups.

Groups are used to make sharing things with friends easier; when sharing something, you can elect to share it with one or more groups, and the item will be shared with all the friends in the selected groups.

Creating groups

To create a new group, tap the add button and enter a name for the new group.

Group details

If you tap on one of the groups, you will be shown a list of friends where you can add or remove friends from the group, rename the group, or delete it.
